# nuxt_amp_demo_2
- Nuxt + AMP

## Servr Setup
```
$ docker-compose build
```

## Server Start
```
$ docker-compose up
```

## Install NPM Module
```
$ docker-compose run --rm server npm i --save MODULE
$ docker-compose run --rm server npm i --save-dev MODULE
```

## How to Deploy
```
$ firebase deploy
```
